📞 TROUBLESHOOTING
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Issue: Cannot connect to database
  → Check DB_PASSWORD in .env.local
  → Verify MySQL is running

Issue: Login doesn't work
  → Did you hash the passwords? (Step 3)
  → Are they in the database? (Step 4)

Issue: File upload fails
  → Do you have BLOB_READ_WRITE_TOKEN set?
  → Get it from Vercel dashboard

Issue: Port 3000 in use
  → npm run dev -- -p 3001 (use different port)

For more help:
  → Read BACKEND_SETUP.md "Troubleshooting" section
