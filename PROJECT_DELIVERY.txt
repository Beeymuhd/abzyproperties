                        SUPPORT & HELP

If Something Isn't Working:

Step 1: Check BACKEND_SETUP.md "Troubleshooting" section
        └─ Common issues with solutions

Step 2: Check TESTING_CHECKLIST.md
        └─ What should work and what doesn't

Step 3: Check terminal for error messages
        └─ Error messages usually explain the problem

Step 4: Check .env.local for missing/wrong values
        └─ Most issues are environment variable related

Step 5: Restart server
        └─ Stop: Ctrl+C
        └─ Start: npm run dev

Common Issues:
- "Cannot connect to database" → Check DB_PASSWORD in .env.local
- "File upload fails" → Set BLOB_READ_WRITE_TOKEN
- "Module not found" → Run npm install
- "Port 3000 in use" → npm run dev -- -p 3001
